Ideas Principales.

  • Id es como un nombre que identifica al elemento.
  • El valor de Id no se puede duplicar

Apuntes de Clase.

Atributo id

El atributo id es como un nombre que identifica al elemento. El valor de este atributo no puede ser duplicado. Este nombre debe ser unico en todo el documento.

Atributo class

Este atributo es mas flexible y puede ser asignado a varios elementos HTML en el documento.

Pseudoclases :nth-child

La Pseudoclase nth-child nos permite encontrar un hijo especifico.

Pseudoclases : nth-child "odd" :nth-child "even"

Las palabras clave odd y even se utilizan para seleccionar elementos con base en su posicion en la secuencia de hijos, especificamente para alternar entre elementos impares y pares.

First-child

Existen otras importantes pseudoclases relacionadas con esta ultima. La pseudoclase First-child referencia solo el primer hijo. Estas pseudoclases en particular no requieren palabras clave o parametros.

Last-Child

Selecciona el último hijo de su elemento padre. Esto significa que, si un elemento es el último hijo de su contenedor

Only-child

La pseudoclase Only-child afecta un elemento siempre y cuando sea el unico hijo disponible.

Not

Not nos permite declarar una Excepcion. Los estilos en la regla creada con esta pseudoclase seran asignados a todo elemento excepto aquellos incluidos en la referencia entre parentesis.

Selector de atributo

Un selector de atributo en css selecciona elementos HTML en funcion de la presencia o el valor de uno de sus atributos. Esto permite aplicar estilos a los elementos que tienen un atributo especifico, o cuyo atributo tiene un valor determinado.

Atributo ^

Una regla con el selector "^" selecciona elementos cuyo atributo comienza con un valor especifico.

Atributo final

Cadena

Una regla con el selector "*" selecciona elementos cuyos atributo contiene una cadena especifica en cualquier posicion.

Relación

Los selectores de relacion en CSS permiten seleccionar elementos basados en su relacion con otros elementos dentro del documento. Estos selectores son utiles para aplicar estilos a elementos que tienen un vinculo estructural con otros elementos, como ancestros, padres, hermanos o descendientes.

Hermanos Adyacentes

Selecciona un elemento que es hermano inmediato, es decir, esta justo despues de otro elemento.

Generales

Selecciona todos los elementos que son hermanos de un elemento anteriror, es decir, esta antes, sin importar si son hermanos inmediatos o no.

Resumen.

En resumen todo esto nos ayuda a que nuestra pagina cuente con un mejor diseño y el usuario pueda tener una mejor interaccion con esta misma.
